==============================================
Derive drupal authorization mapping from DN (FromDN)
-----------------------
setup: configure a DN based mapping to a consumer type (drupal role).
  check drupal role is derived from user's DN and pick an attribute such a "ou"
-----------------------
Tests and Desired Results
-----------------------

LDAP_authorz.FromDN.nomatch  -- user authenticates and has no ou match in ldap object
LDAP_authorz.FromDN.manymatch  -- many matches on dn attribute.  what should happen?  mulitple roles mapped!
LDAP_authorz.FromDN.onematch  -- single match.  provide match to authorization child module.
